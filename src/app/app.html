Game Score Cards

Create customizable score cards for your games

Instructions

Game Creation

  1. To start a new game, enter a game name and add at least one player or team.
  2. Use the button to add players/teams.
  3. When ready, click the "Start Playing" button to create the game and begin adding rounds and scores.
  4. Existing games are listed in the "Existing Games" section. Click the or buttons to view a paused game or continue a game. Click the button to manage players or delete a game.

Game Play

  1. Click the button to return to the game creation screen
  2. Click the button to finish/pause a game and the button to resume a paused game.
  3. Add a round by entering the scoring item name (e.g. Hand 1 (for a hand in Euchure or Bridge) or Military (for the red military count in Seven Wonders)) and clicking the button. This will create a group containging the names of each player/team.
  4. You can sort the scoring groups by clicking the or buttons. By default, newly added scoring groups are added to the top of the game score card.
  5. To remove a scoring group, click the button next to the scoring group name.
  6. Enter the score for each player/team in the newly created round. Scores can be positive or negative numbers.
  7. The total scores will automatically update as you enter scores for each round.
